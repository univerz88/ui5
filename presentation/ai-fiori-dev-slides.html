MCP Tools
Model Context Protocol

MCP = Claude의
UI5 전문 지식 연결

새 프로젝트 시작
get_guidelines

UI5 개발 원칙 자동 확인

create_ui5_app

TypeScript 프로젝트 구조 생성

get_api_reference

VizFrame, FileUploader 등 API 확인

run_ui5_linter

코드 품질 및 표준 준수 검사

이게 왜 중요한가

UI5 경험 없이도 가능
MCP가 최신 UI5 가이드라인을 자동으로 제공하므로 Claude가 표준에 맞는 코드를 생성함
오류 시 원인 설명 포함
단순 코드 수정이 아닌 "왜 이렇게 해야 하는지" 이해하며 진행
프로토타입이 명세서 역할
demo.html이 있어 요구사항이 명확 → AI가 정확하게 구현
